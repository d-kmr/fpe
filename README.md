# fpe
Function Pointer Eliminator

[Requirements]
- gcc
- cmake
- zlib
- git
- xcode (macOS)
- tcsh or bash
- z3

[Opam requirement]
- Ocaml 4.05.0
- ocamlfind
- ppx_deriving_yojson

[Installation]
1. git clone https://github.com/d-kmr/fpe.git
2. cd fpe
3. ./fpebuild make

[Use]
- ./fpe <C-Project-Directory>
